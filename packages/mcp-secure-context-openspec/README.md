# @mcp-secure-context/openspec

OpenSpec is the standards-facing contract layer for MCP Secure Context Sharing.

This package provides:

- umbrella exports for `@openspec/schema`
- umbrella exports for `@openspec/runtime`
- Secure Context container validation/digest helpers
- a single Secure Context Sharing import path for consumers that prefer the `@mcp-secure-context/*` family
