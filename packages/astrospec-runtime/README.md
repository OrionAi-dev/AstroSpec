# @astrospec/runtime

Runtime validators, locking, diffing, merging, and verification for AstroSpec core contracts.

## Scope

This package implements the core runtime layer for:

- schema validation
- deterministic locking and signing
- context and turn merge and diff
- verifier execution

It is part of the AstroSpec core.

## Use It When

Use this package when you want the implementation layer behind AstroSpec core validation, not just the raw schema files.

## Commands

```bash
pnpm -F @astrospec/runtime build
pnpm -F @astrospec/runtime test
```
