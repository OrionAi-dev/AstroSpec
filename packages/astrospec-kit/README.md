# @astrospec/kit

Small DX layer for AstroSpec validation and MCP tool calls.

## Scope

This package is not the core standard surface. It is a convenience layer for consumers who want:

- simple validation helpers
- retrieval-profile validation helpers
- simple MCP tool invocation helpers

## Use It When

Use this package for small consumer integrations, prototypes, and adapter code that should stay close to the standard but does not need to wire the full runtime directly.
