# @astrospec/cli

CLI tooling for AstroSpec core and retrieval-profile contracts.

## Commands

- `astrospec validate`
- `astrospec lock`
- `astrospec merge`
- `astrospec diff`
- `astrospec verify`
- `astrospec validate-contract`
- `astrospec doctor`

## Use It When

Use the CLI when you need a fast schema-first validation path, local contract inspection, or a smoke-level adoption check for payloads and integrations.
