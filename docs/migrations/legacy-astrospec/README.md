# Legacy AstroSpec Migration

This repo was previously positioned around AstroSpec.

The current public identity is **MCP Secure Context Sharing**, with **OpenSpec** as the canonical standards family.

## Removed AstroSpec names

The following package names were retired as part of the cutover:

- `@astrospec/schema`
- `@astrospec/runtime`
- `@astrospec/mcp-profile`
- `@astrospec/kit`
- `@astrospec/cli`

## Current mapping

- `@astrospec/schema` -> `@openspec/schema`
- `@astrospec/runtime` -> `@openspec/runtime`
- `@astrospec/mcp-profile` -> `@mcp-secure-context/mcp-adapter`
- `@astrospec/kit` -> `@openspec/kit`
- `@astrospec/cli` -> `@openspec/cli`

## Legacy docs

Historical AstroSpec notes now live only in migration and legacy documentation.
